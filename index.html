App Gimnasio

Esta aplicacion fue realizada
como proyecto personal,
a modo de practica
combinando mis actividades favoritas,
la programacion y el gimnasio.
En la App Web busco resolver el
problema de la mayoria de los gimnasios,
de no saber cual es la rutina
que te corresponde.
La idea es asignarle una rutina a un alumno
que pueda entrar a la aplicacion
y ver los ejercicios que le corresponde cada dia.

Tecnologias utilizadas:

Java VueJS Spring Boot PostgreSQL Docker
