💻 Proyectos Destacados

App Web para Taller Mecanico

Esta aplicacion fue realizada como proyecto final en la universidad. Desarrolle este Sistema con el fin de resolver el problema de gestion y control del inventario del taller mecanico. Tambien se permite agendar turnos, crear presupuestos, registrar vehiculos, clientes, pedidos, productos e insumos. La aplicacion se desplego mediante Docker y esta siendo utilizada por el Taller, a quienes brinde una capacitacion sobre el uso de la misma para que la puedan utilizar correctamente

App Web Gimnasio

Actualmente, estoy desarrollando esta aplicacion a modo de practica, combinando mis actividades favoritas, la programacion y el gimnasio. Lo que busco es resolver el problema de la mayoria de los gimnasios, de no saber cual es la rutina que te corresponde, los ejercicios que se deben realizar por dia. La idea es asignarle una rutina a un alumno que pueda entrar a la aplicacion y ver los ejercicios que le corresponde cada dia.
